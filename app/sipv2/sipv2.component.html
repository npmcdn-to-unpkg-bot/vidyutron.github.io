Social Intelligence Platform V2

  • Client: Microsoft
  • Date: March 2016-July 2016
  • Service: Web Development

Project Description

SIP, is platfrom for carrying analytics and monitoring of Social media concensous, related to microsoft products and their competitors from major platforms like blogs,twitter,facebook etc.

As the volume to data flowing was huge, generic system has to be migrated to faster and adaptive BI mediums like Cosmos, Elastic Search, Azure No SQL storage.

At the end of the data flows lies the UI, which is responsive both visually and functionally in order to handle massive incoming data and perform real time rendering of Graphs, documents, filters etc.
